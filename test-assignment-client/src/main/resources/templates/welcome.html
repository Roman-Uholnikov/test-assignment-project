Simple search engine

This is simple search engine. You can add document to be used in search. Also you can search for particular phrase in stored documents.

Search principles

  • exact match means that we want to find exact matching of the phrase in the document (prefix and postfix are not counted)
  • if no exact matching selected, then match: all documents that contain all words from he phrase (words should match exactly, no endings or prefixes), words from phrarse should be found in the order they were provided.

Example 1:

Search word: interesting test assignment

Match:

We search for interesting people. We test them a lot. After they receive assignment.

However Example 2:

Search word: interesting test assignment

No match:

We send them interesting test assignments
